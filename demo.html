Available Features

  • Browse complete menu with categories
  • Add items to cart with quantities
  • Place orders with special instructions
  • Call waiter with different request types
  • View order history for the table
  • Cancel orders within 1-minute window
  • Real-time countdown timer for cancellation
  • Session-based table locking
  • XSS protection and input sanitization
Production Mode

In production, customers MUST scan QR codes to access the menu. Demo mode is for testing purposes only and should be disabled before deployment.
